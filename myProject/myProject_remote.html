Simon Memory Game

GAME RULES (PLEASE READ!)

Follow each step in order (1, 2, 3)

If you see 'YES!' after Step 3: Repeat Step 2

If you see 'NO!' after Step 3: Repeat Step 1 and play again

1. Generate New Sequence

2. Show Sequence

3. Guess Sequence

G for Green, R for Red, Y for Yellow, B for Blue
